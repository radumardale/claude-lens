Quick Start

Run directly (recommended)

$ npx claude-lens

Or install globally

$ npm install -g claude-lens
$ claude-lens

CLI commands

$ claude-lens scan # Text summary
$ claude-lens scan --json # JSON output
$ claude-lens disable mcp rollbar
$ claude-lens enable plugin figma
